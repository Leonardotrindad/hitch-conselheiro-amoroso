Hitch - Conselheiro Amoroso

O Hitch é uma Inteligência Artificial conselheira amorosa desenvolvida para ajudar usuários em situações complexas de relacionamento. O sistema fornece conselhos personalizados baseados em análise de sentimentos e padrões comportamentais.

Tecnologias Utilizadas

- Backend: Java 21, Spring Boot (Web, Data JPA, Security, Validation)
- Banco de Dados: H2 Database (Dev) / PostgreSQL (Prod)
- Documentação: Swagger / Google.genai
- Frontend: HTML5, CSS3, JavaScript (JSX), React e Talwind.

Como Rodar o Projeto

1. Clone este repositório.
2. Abra o projeto na sua IDE favorita (VS Code, IntelliJ).
3. Execute a classe principal HitchApplication.java.
4. Acesse o Swagger UI para testar as APIs:
   http://localhost:8080/swagger-ui/index.html
